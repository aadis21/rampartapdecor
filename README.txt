# Rampratap Enterprises LLP — Website

## Files
- index.html  — Main HTML
- style.css   — All styles (fully responsive)
- script.js   — Interactions (mobile nav, scroll reveal, form)

## How to Use
1. Open index.html in any browser — no build step needed.
2. To host: upload all 3 files to any web server or hosting service.
3. Images are loaded from Unsplash CDN — replace URLs with your own photos as needed.

## Fonts
Loaded from Google Fonts (Cormorant Garamond, Cinzel, Raleway).
Requires internet connection unless you self-host the fonts.

## Customisation
- Colors: edit CSS variables at the top of style.css
- Contact details: search for "+91-9354002700" in index.html
- Brand descriptions: find each .brand-card block in index.html
